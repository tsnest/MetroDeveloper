[RU]:

Данный проект частично восстанавливает некоторые возможности разработчиков в движке 4A Engine.

Возможности:

1. Отключение вывода сообщения о предыдущем неудачном запуске игры.
2. Отключение интро роликов при запуске игры.
3. Разблокировка консоли разработчика.
4. Частичная разблокировка чтения ресурсов из папки content.
5. Возможность использования dds текстур в last light и более новых играх.
6. Восстановление консольных команд во всех играх (g_god, g_global_god, g_notarget, g_unlimitedammo, g_unlimitedfilters, fly, refly).
7. Восстановление консольной команды r_hud_weapon в Last Light и в Redux.
8. Восстановление консольной команды r_base_fov в непропатченной версии Last Light.
9. Восстановление консольной команды g_godbless в Exodus и Arktika.1.
10. Возможность включения режима от третьего лица (данный режим имеет баги).
11. Быстрые сохранения.
12. Режим полёта и телепорта.
13. Клаво-мышные патчи для Arktika.1.
14. Генерация nav_map.pe или nav_map.bin.
15. Отключение повышения цен на фильтры в redux.
16. Раскачка оружия из Lost Alpha для оригинальной Metro 2033.
17. Подмена стандартного шрифта в консоли и в режиме stats для оригинальной Metro 2033
18. Вывод сообщений о ненайденных файлах в консоль и в лог
19. Возможность включения BugTrap


[EN]:

This project partially restores some of the developer features in the 4A Engine.

Features:

1. Disabled the message about the previous failed game launch.
2. Disabled intro videos when launching the game.
3. Unlocked the developer console.
4. Partially unlocked the ability to read resources from the content folder.
5. Added support for DDS textures in Last Light and newer games.
6. Restored console commands in all games (g_god, g_global_god, g_notarget, g_unlimitedammo, g_unlimitedfilters, fly, refly).
7. Restored the r_hud_weapon console command in Last Light and Redux.
8. Restoration of the r_base_fov console command in the unpatched version of Last Light.
9. Restoration of the g_godbless console command in Exodus and Arktika.1.
10. Ability to enable third-person mode (this mode has bugs).
11. Quick saves.
12. Flight and teleport modes.
13. Keyboard and mouse patches for Arktika.1.
14. Generation of nav_map.pe or nav_map.bin.
15. Disabling price increases for filters in Redux.
16. Weapon bobbing from Lost Alpha for the original Metro 2033.
17. Replacement of the default font in the console and stats mode for the original Metro 2033.
18. Display of messages about missing files in the console and log.
19. Option to enable BugTrap.
